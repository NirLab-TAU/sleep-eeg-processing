plot_psd_per_stage(picks=('E101',), psd_range=(-40, 60), freq_range=(0, 40), xscale='linear', sleep_stages={'N1': 1, 'N2': 2, 'N3': 3, 'REM': 4, 'Wake': 0}, axis=None, plot_sensors=False, save=False, psd_method_args=None, subplots_args=None)[source]#

Plot PSD per sleep stage.

Parameters:

  • picks (str | Iterable[str]) – Channels to calculate PSD on, more info at mne.io.Raw.get_data. Defaults to (“E101”,).

  • psd_range (tuple) – Range of y axis on PSD plot. Defaults to (-40, 60).

  • freq_range (tuple) – Range of x axis on PSD plot. Defaults to (0, 40).

  • xscale (str) – Scale of the X axis, check available values at matplotlib.axes.Axes.set_xscale. Defaults to “linear”.

  • sleep_stages (dict) – Mapping between sleep stages names and their integer representations. Defaults to {“Wake”: 0, “N1”: 1, “N2”: 2, “N3”: 3, “REM”: 4}.

  • axis (Optional[axis]) – Instance of matplotlib.pyplot.axis. Defaults to None.

  • plot_sensors (bool) – Whether to plot sensor map showing which channels were used for computing PSD. Defaults to False.

  • save (bool) – Whether to save the figure. Defaults to False.

  • psd_method_args (Optional[dict]) – Arguments passed to the PSD method, e.g., welch. Defaults to None.

  • subplots_args (Optional[dict]) – Arguments passed to the plt.subplots(). Have no effect if axis is provided. Defaults to None.

Plots topomap for a sleep stage and a frequency band.

Parameters:

  • stage (str) – One of the sleep_stages keys. Defaults to “REM”.

  • band (dict) – Name-value pair - with name=arbitrary name and value=(l_freq, h_freq). Defaults to {“Delta”: (0, 4)}.

  • dB (bool) – Whether transform PSD to dB. Defaults to False.

  • sleep_stages (dict) – Mapping between sleep stages names and their integer representations. Defaults to {“Wake”: 0, “N1”: 1, “N2”: 2, “N3”: 3, “REM”: 4}.

  • axis (Optional[axis]) –

    Instance of matplotlib.pyplot.axis. Defaults to None.

  • fooof (bool) – Whether to plot parametrised spectra. More at fooof. Defaults to False.

  • save (bool) – Whether to save the figure. Defaults to False.

  • psd_method_args (Optional[dict]) – Arguments passed to the PSD method, e.g., welch. Defaults to None.

  • topomap_args (Optional[dict]) – Arguments passed to mne.viz.plot_topomap(). Defaults to None.

  • cbar_args (Optional[dict]) – Arguments passed to plt.colorbar(). Defaults to None.

  • fooof_group_args (Optional[dict]) – Arguments passed to fooof.FOOOFGroup(). Defaults to None.

  • fooof_get_band_peak_fg_args (Optional[dict]) – Arguments passed to the fooof.analysis.get_band_peak_fg(). Defaults to None.

  • subplots_args (Optional[dict]) – Arguments passed to the plt.subplots(). Have no effect if axis is provided. Defaults to None.

plot_topomap_collage(stages_to_plot='all', bands={'Alpha': (8, 12.49), 'Beta': (12.5, 29.99), 'Delta': (0, 3.99), 'Gamma': (30, 60), 'SMR': (12.5, 15), 'Theta': (4, 7.99)}, dB=False, sleep_stages={'N1': 1, 'N2': 2, 'N3': 3, 'REM': 4, 'Wake': 0}, fooof=False, low_percentile=5, high_percentile=95, save=False, psd_method_args=None, topomap_args=None, cbar_args=None, fooof_group_args=None, fooof_get_band_peak_fg_args=None, figure_args=None)[source]#

Plots topomap collage for multiple sleep stages and bands.

Parameters:

  • stages_to_plot (tuple) – Tuple of strings representing names from sleep_stages, e.g., (“REM”, “N1”). If set to “all” plots every stage provided in sleep_stages. Defaults to “all”.

  • bands (dict) – Dict of name-value pairs - with name=arbitrary name and value=(l_freq, h_freq). Defaults to { “Delta”: (0, 3.99), “Theta”: (4, 7.99), “Alpha”: (8, 12.49), “SMR”: (12.5, 15), “Beta”: (12.5, 29.99), “Gamma”: (30, 60), }.

  • dB (bool) – Whether transform PSD to dB. Defaults to False.

  • sleep_stages (dict) – Mapping between sleep stages names and their integer representations. Defaults to {“Wake”: 0, “N1”: 1, “N2”: 2, “N3”: 3, “REM”: 4}.

  • fooof (bool) –

    Whether to plot parametrised spectra. More at fooof. Defaults to False.

  • low_percentile (float) – Set min color value by percentile of the band data. Defaults to 5.

  • high_percentile (float) – Set max color value by percentile of the band data. Defaults to 95.

  • save (bool) – Whether to save the figure. Defaults to False.

  • psd_method_args (Optional[dict]) – Arguments passed to the PSD method, e.g., welch. Defaults to None.

  • topomap_args (Optional[dict]) –

    Arguments passed to mne.viz.plot_topomap(). Defaults to None.

  • cbar_args (Optional[dict]) –

    Arguments passed to plt.colorbar(). Defaults to None.

  • fooof_group_args (Optional[dict]) –

    Arguments passed to fooof.FOOOFGroup(). Defaults to None.

  • fooof_get_band_peak_fg_args (Optional[dict]) –

    Arguments passed to the fooof.analysis.get_band_peak_fg(). Defaults to None.

  • figure_args (Optional[dict]) – Arguments passed to plt.figure(). Defaults to None.
