ICAPipe#

class sleepeeg.pipeline.ICAPipe(prec_pipe=None, path_to_eeg='', output_dir='', method='fastica', n_components=None, fit_params=None, path_to_ica=None, **ica_kwargs)[source]#

Bases: BasePipe

The ICA pipeline element.

Contains ica fitting, plotting multiple ica plots, selecting ica exclusion components and its application to the raw data. More at mne.preprocessing.ICA.
