class sleepeeg.pipeline.ICAPipe(*, prec_pipe=None, path_to_eeg=_Nothing.NOTHING, output_dir=_Nothing.NOTHING, n_components=30, method='fastica', fit_params=None, path_to_ica='/', mne_ica=_Nothing.NOTHING)[source]#

Bases: BasePipe

The ICA pipeline element.

Contains ica fitting, plotting multiple ica plots, selecting ica exclusion components and its application to the raw data. More at mne.preprocessing.ICA.

n_components: UnionType[int, float, None]#

Number of principal components (from the pre-whitening PCA step) that are passed to the ICA algorithm during fitting.

method: str#

The ICA method to use in the fit method.

Can be ‘fastica’, ‘infomax’ or ‘picard’ Use the fit_params argument to set additional parameters.

fit_params: dict#

Additional parameters passed to the ICA estimator as specified by method. Allowed entries are determined by the various algorithm implementations: see FastICA, picard and infomax.

plot(save_annotations=False, save_bad_channels=False, overwrite=False, mne_plot_args=None)#

A wrapper for mne.io.Raw.plot().

Parameters:

  • butterfly – Whether to start in butterfly mode. Defaults to False.

  • save_annotations (bool) – Whether to save annotations as txt. Defaults to False.

  • save_bad_channels (bool) – Whether to save bad channels as txt. Defaults to False.

  • scalings – Scale for amplitude per channel type. Defaults to ‘auto’.

  • use_opengl – Whether to use OpenGL acceleration. Defaults to None.

  • overwrite (bool) – Whether to overwrite annotations and bad_channels files if exist. Defaults to False.

  • mne_plot_args (Optional[dict]) – Arguments passed to raw.plot(). Defaults to None.
